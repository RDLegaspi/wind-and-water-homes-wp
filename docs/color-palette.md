## Additional Theme Families

The live Wix theme also includes these secondary families:

- Blue: `#B8E3F9`, `#94D2F3`, `#34ADED`, `#23739E`, `#113A4F`
- Green: `#C6EFD9`, `#A6DFC0`, `#65CF95`, `#438A63`, `#224532`

The current WordPress theme maps the primary navy, red, white, gray, and pale blue colors into CSS custom properties in `style.css`.
